Functions

1. Intitializing

To initialize, we need 2 things: the ticker symbol of the company of interest, and the time period in which the data comes from. Time Periods include: "Xmo", "Xd", "Xy", "ytd" (Year-To-Date), and "max" (All Time), where X is an integer. This function also initialized a few things and sets them to the value of None. This included data that we use later on.

2. get_data(self, data_type=None)

This function is meant to retrieve the stock data that you want. This is done through Yahoo finance's API. What this function takes is simply a data type. This data type being what you want to look for in the stock data (High, Low, Open, etc.)

3. model_data(self, column)

This function is supposed to model the real data we get in a time series plot. All it takes is the specific column of data from the get_data function. NOTE: This function does not work if you do not choose a specific aspect of stock valuation (such as Open, High, Low, etc.)

4. train_model(self)

This function uses the data we get from get_data and trains a prophet model on it. This is what we will use to predict our stock valuation.

5. predict(self, steps)

This function is what we use to actually predict the stock prices. All it takes is the number of steps (this is in the unit of whatever was chosen when intitializing). This also utilized the same Prophet model created in train_model.

6. plot_forecast(self, forecast)

This function plots the original data from get_data AND plots the forecasted/predicted data frp, predict. Again, in a time-series-like plot.

7. error(self, forecast)

This function calculates the error in our predictions.
