What is it exactly?

This mini-project is a means of (roughly) predicting stock valuation for a company.

It uses Yahoo Finance's API to gather the data on a companies stock valuation whether it be high, low, open, etc.

Within the program itself, it can model the data that the API gets, predict what happens next through the use of ARIMA.

More information will be given later in this process.

Why?
